# Satellite Imagery for Tracking Flood Changes Over Time: A Case Study of the Pamba River Basin

## Overview
This project focuses on flood inundation and flood susceptibility mapping in the Pamba River Basin during the 2018 Kerala floods using Sentinel-1 SAR imagery, GIS techniques, and Google Earth Engine (GEE).

## Objectives
- Analyze pre-flood and post-flood SAR imagery
- Detect and map flood inundation
- Assess flood susceptibility zones
- Integrate GIS, Sentinel-1 SAR, and GEE for flood analysis
- Support flood preparedness and disaster management

## Study Area
Pamba River Basin, Kerala, India

## Data Used
- Sentinel-1 SAR Data
- DEM (SRTM 30m)
- Rainfall Data
- LULC Data
- Geological and Hydrological Layers

## Software & Tools
- Google Earth Engine (GEE)
- ArcGIS
- GIS Techniques
- Remote Sensing

## Methodology
The study used Sentinel-1 SAR imagery to compare pre-flood and post-flood conditions during the 2018 Kerala floods. Flood inundation mapping and susceptibility analysis were performed using GIS-based weighted overlay methods.

## Key Outputs
- Flood Inundation Map
- Flood Susceptibility Map
- GIS-Based Spatial Analysis
- Flood Preparedness Framework
